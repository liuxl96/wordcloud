# wordcloud
将中英文的文本数据生成可视化的词云，并且可根据背景图片定制词云的形状。
## 编译环境
+ Python 3
## 需安装的包
+ numpy
+ matplotlib
+ wordcloud
+ imageio
+ jieba
+ re
+ PIL
